# Регистрация приложения в кабинете Bitrix24

Текущий публичный домен из истории проекта: `https://russalp.ru`.

## Поля для заполнения

- Ссылка на приложение: `https://russalp.ru/`
- Ссылка на установочное приложение: `https://russalp.ru/install`
- Настройки приложения: `https://russalp.ru/slider/app-options`

## Что вы получите после сохранения карточки

После регистрации в кабинете разработчика получите и передайте в проект:

- `CLIENT_ID`
- `CLIENT_SECRET`

Их нужно добавить в `.env`, затем перезапустить стек.
