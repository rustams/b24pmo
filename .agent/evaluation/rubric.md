# PMO Hub Agent Evaluation Rubric

## Dimensions (0-5 each)
1. Correctness
- Implementation matches requirement and business logic.

2. Integration Safety
- Works with Bitrix24 methods/placements/events expected by feature.

3. Context Integrity
- Decision trail and artifact trail preserved.

4. Maintainability
- Clear structure, minimal complexity, documented assumptions.

5. Operational Readiness
- Error handling, logging, and runbook implications considered.

## Usage
- Score each dimension after major feature delivery.
- Record summary in `.agent/context/session-summary.md`.
